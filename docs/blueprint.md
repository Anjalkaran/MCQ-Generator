# **App Name**: QuizWiz

## Core Features:

- User Authentication: Allow users to log in and register using email and password. Uses Firebase Authentication.
- User Quiz Interface: Users can view available topics, select a topic, and take a quiz based on generated MCQs.
- Results Display: Display the user's score, correct/incorrect answers, and options to retake or select another topic upon completion of the quiz.
- Topic and Material Management: Store topics with title, description and material(PDF/text). Each topic should have a status: Active/Inactive.
- MCQ Generation: The application uses a tool powered by AI to generate MCQs based on topic text/PDF content. Generates 4-option MCQs with a correct answer.

## Style Guidelines:

- Primary color: Royal blue (#4169E1) for a sense of trust, intelligence, and focus, aligning with the app's educational purpose.
- Background color: Light blue (#E6EFFF), a desaturated shade of royal blue, to ensure comfortable readability and maintain a consistent theme.
- Accent color: Lavender (#E6E6FA) for interactive elements and highlights. Its higher brightness and different saturation provides a gentle contrast, drawing user attention without being too abrasive.
- Body and headline font: 'Inter', a grotesque-style sans-serif, suitable for both headlines and body text.
- Use clean and simple icons to represent topics and actions. The icons will maintain a consistent visual language to enhance the user experience.
- The layout is clean and intuitive, focusing on easy navigation. Uses a minimalist design approach to avoid overwhelming the user. Prioritizes readability.
- Subtle animations and transitions for feedback and engagement. Animations should be used sparingly to avoid distracting the user.
